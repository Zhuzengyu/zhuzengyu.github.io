服务器规则

为了维护良好的游戏环境，请遵守以下规则

查看规则

重要提示

所有玩家在加入服务器前必须阅读并同意遵守以下规则。违反规则将根据情节严重程度受到相应处罚，包括但不限于警告、禁言、封禁等。

规则最终解释权归服务器管理团队所有。

基本规则

所有玩家必须遵守的基本原则

行为规范

玩家行为基本准则

  • 禁止使用任何外挂、作弊客户端 严重
  • 禁止恶意刷屏、辱骂他人 中等
  • 禁止冒充管理员或其他玩家 严重
  • 尊重其他玩家，文明游戏 轻微

服务器规则

服务器运行相关规则

  • 禁止恶意攻击服务器 严重
  • 禁止利用游戏漏洞 中等
  • 禁止使用恶意插件 严重
  • 发现BUG请及时报告 轻微

游戏规则

游戏内具体行为规范

建筑规则

建筑与领地相关规则

  • 禁止建造不雅、违规建筑 中等
  • 禁止恶意圈地阻碍交通 中等
  • 禁止破坏他人建筑 严重
  • 砍树请清理干净 轻微

PVP规则

玩家对战相关规则

  • PVP区域禁止恶意蹲守 中等
  • 安全区禁止PVP 严重
  • 决斗需双方同意 轻微
  • 禁止使用PVP外挂 严重

社区规则

玩家交流与社区互动规则

聊天规则

聊天与交流规范

  • 主要使用中文交流 轻微
  • 禁止政治敏感话题 严重
  • 正确使用频道功能 轻微
  • 禁止泄露他人隐私 严重

交易规则

玩家间交易规范

  • 交易需双方自愿 中等
  • 禁止现实货币交易 严重
  • 禁止欺诈行为 严重
  • 建议使用交易系统 轻微

处罚说明

违规行为对应的处罚措施

处罚等级

根据违规严重程度分级

  • 轻微违规：警告、短期禁言
  • 中等违规：物品没收、中期封禁
  • 严重违规：永久封禁、IP封禁
  • 管理团队有权根据具体情况调整处罚

申诉流程

对处罚有异议的申诉方式

  • 通过QQ群联系管理员
  • 提供详细的情况说明
  • 提供相关证据材料
  • 申诉处理需要1-3个工作日

遵守规则，快乐游戏
